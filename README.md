# Tres en Raya

Proyecto de ejemplo: juego Tic-Tac-Toe (tres en raya) para la asignatura.

Contenido del repositorio:
- `index.html` — Interfaz principal
- `assets/` — imágenes y recursos
- `css/styles.css` — estilos
- `js/game.js` — lógica del juego
- `js/storage.js` — almacenamiento/localStorage

Cómo subir a GitHub (si ya tienes remoto):
1. git remote add origin <URL-REMOTE>
2. git branch -M main
3. git push -u origin main

Si quieres que cree el repo remoto automáticamente usa la GitHub CLI (`gh`) y estaré encantado de intentarlo.
